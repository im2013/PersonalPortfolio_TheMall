Creature Creator

An application where I wish to create animals or other organisms from scratch. I am making this application using C# within the Unity Engine, and all 3D modelling is done within Blender. The aim of this app is to be able to save, load, export (as FBX files) completely new animals and creatures. I hope to make this creature creator very customisable with the help of UI, blend shapes and bezier curves. The animal created is to have a full anatomy including a skeleton, muscles, a basic organ system, skin and fat.
